# JapaneseTown
A small interactive Japanese town made in Unity.

[Demo Video](https://youtu.be/4Lk3K_R4-LA)

* [Fantasy Skybox FREE](https://assetstore.unity.com/packages/2d/textures-materials/sky/fantasy-skybox-free-18353)
